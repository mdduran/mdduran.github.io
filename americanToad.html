American Toad

Year Developed: 2015

Genre: Solitaire Card Game

Purpose and Accomplishments: This game was an individual assignment for the CS 3733 "Software Engineering" course taught at WPI. The excercise required students to design a game with specific rules using a given framework of code in Java. The framework was developed by George Hieneman. Using the solitaire framework and the E(entity)B(boundary)C(control) model for software enginerring practices, I created a solitaire game that reached 85.3 percent code coverage.

Splash Image:

Game Description: American Toad is a solitaire game using two decks of playing cards. This game is similar to Canfield except that the Tableau builds down in suit, and a partial Tableau Stack cannot be moved (only the top card or entire stack can be moved). The object of the game is to move all cards to the Foundations.Source
